Building Trustworthy Charity Platforms in Kenya: A Decentralised Approach with DAO

Technologies Used

Blockchain Smart Contracts React Node.js Express JavaScript TypeScript Solidity Web3.js IPFS

Project Description

This dissertation project addresses the critical challenge of trust in charitable organizations in Kenya by leveraging blockchain technology and decentralized autonomous organization (DAO) principles.

The platform enables transparent tracking of donations, ensures accountability through smart contracts, and empowers donors to participate in decision-making processes. By implementing a DAO structure, the project creates a self-governing ecosystem where stakeholders can propose, vote on, and execute charitable initiatives.

Key features include immutable transaction records, real-time fund tracking, democratic governance mechanisms, and integration with local financial systems to ensure accessibility for Kenyan users.

Project Outcomes

  • Increased transparency in charitable donations and fund utilization
  • Enhanced donor confidence through verifiable impact tracking
  • Reduced administrative overhead and operational costs
  • Created a model for democratic governance in charitable organizations
  • Established a framework that can be adapted for other regions facing similar challenges

Electronic Voting Innovation Project

Technologies Used

Web3 Truffle Ganache JavaScript MetaMask NodeJS Ethereum

Project Description

The Electronic Voting Innovation Project was developed to address the challenges of traditional voting systems, including security concerns, lack of transparency, and inefficiency.

By leveraging blockchain technology, this system ensures that votes are immutable, transparent, and verifiable while maintaining voter privacy. The solution includes a user-friendly interface for voters and administrators, secure authentication mechanisms, and real-time result tabulation.

The project demonstrates how blockchain can revolutionize democratic processes by eliminating the possibility of fraud, reducing costs, and increasing accessibility for voters.

Project Outcomes

  • Created a tamper-proof voting system with cryptographic security
  • Enabled transparent vote counting while preserving voter anonymity
  • Reduced the cost and logistical complexity of election administration
  • Increased voter participation through improved accessibility
  • Developed a scalable solution that can be adapted for various election types
